## Project overview
SQL Portfolio project analyzes customer orders and Health Kits sales using SQL.
## Skills Demonstrated
* SQL Joins
* Aggregate Functions(SUM,COUNT)
* GROUP BY
* Stored Procedures
* Window Functions
* Data Analysis
## Database Tables
### Customers
Contains customer information.
### Health Kits
Contains Health Kit details and pricing.
### Customer Orders
Contains customer purchases and order dates.
### Analytical Queries
Analyzes customer ordering patterns and revenue.
### Window Function Example
Uses ROW_NUMBER() to assign an order sequence number for each customer's purchases.
### Stored Procedure Example
This is used to insert new customer orders into the database.
